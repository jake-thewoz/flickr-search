Flickr Tag Search (using AJAX)

Treehouse

This project was done along with a tutorial from Treehouse. They provided the styling. This was an excercise to ues both JSON and AJAX to make an image search program.

The Project

This was a fun and rewarding project that guided me through the process of AJAX. The program passes information to and from the Flickr API, so that it returns the most recent Flickr pictures with the search tag. Try it out! The data format is JSON. The CSS is a little heavy (and I didn't write it), but it's extremely functional and responsive.
